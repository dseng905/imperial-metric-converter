Imperial to Metric Converter

Overview

Convert imperial units such as gallons and miles to their respective metric units by sending a GET request to JSON file containing the original and converted unit.


For example, type the following URL [app URL]/api/convert?input=3.1gal to get the following JSON object:
{"initNum":3.1,"initUnit":"mi","returnNum":4.98895,"returnUnit":"km","string":"3.1 miles converts to 4.98895 kilometers"}

Use the following abbreviated units: gal, lbs, and mi, to convert to liters, kilograms, and kilometers respectively.
